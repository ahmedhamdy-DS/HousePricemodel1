# 🏡 House Price Prediction (Kaggle Project)

This project aims to predict house prices based on various features using multiple regression models.

## 📊 Dataset
- Source: [Kaggle Competition](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques)
- Train size: 1460 samples
- Test size: 1459 samples

## 🔧 Tools & Libraries
- Python
- pandas, numpy
- seaborn, matplotlib
- scikit-learn (LinearRegression, RandomForest, GradientBoosting)

## 🧪 Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## ✅ Best Score
- **Model:** RandomForestRegressor()
- **R² Score on validation set:** 0.97

## 📦 Files in This Repo
- `house_price_model.ipynb`: Full notebook with data cleaning, modeling, and evaluation.
- `submission.csv`: Prediction file for Kaggle submission. _(optional)_

## 📌 How to Run
1. Install required libraries
2. Download the dataset from Kaggle
3. Run the Jupyter notebook
